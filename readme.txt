Ultra-lightweight bundle products plugin optimizado para sitios médicos y Blocksy Pro.

== Description ==

SimMedical Smart Bundles es un plugin ultraliviano diseñado específicamente para crear paquetes de productos médicos en WooCommerce. Optimizado para funcionar perfectamente con Blocksy Pro.

**Características principales:**

* ✅ Suma automática de precios de productos
* ✅ Precio manual rebajado para packs
* ✅ Cálculo automático de descuentos
* ✅ Interface admin intuitiva
* ✅ Display frontend optimizado
* ✅ Compatible con Blocksy Pro
* ✅ Terminología médica específica
* ✅ Ultra-liviano (~15KB total)

**Ideal para:**
* Consultorios médicos
* Clínicas
* Distribuidores de equipos médicos
* Tiendas de suministros médicos

== Installation ==

1. Sube el plugin a `/wp-content/plugins/simmedical-bundles/`
2. Activa el plugin desde 'Plugins' en WordPress
3. Ve a Productos → Agregar nuevo
4. Marca "Es un Bundle Médico" en Datos del producto
5. Selecciona productos del bundle
6. Define precio rebajado
7. Publica el producto

== Frequently Asked Questions ==

= ¿Es compatible con otros temas? =
Sí, aunque está optimizado para Blocksy Pro, funciona con cualquier tema compatible con WooCommerce.

= ¿Afecta la performance del sitio? =
No, el plugin es ultra-liviano (~15KB) y está optimizado para máxima performance.

= ¿Puedo personalizar los estilos? =
Sí, puedes personalizar los estilos desde tu tema child o desde Customizer.
